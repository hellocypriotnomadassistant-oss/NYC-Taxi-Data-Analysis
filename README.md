# NYC Taxi Data Analysis 🚕

## Project Overview
This project involves an Exploratory Data Analysis (EDA) of the New York City taxi trip dataset. The primary goal is to assess data quality, handle missing values, and uncover patterns related to taxi fares and trip distances.

## Key Insights
- **Data Preparation:** Cleaned the raw taxi trip data by addressing null values and inconsistencies.
- **EDA Findings:** Identified correlations between pickup/dropoff times and fare amounts.
- **Strategy:** Followed the PACE (Plan, Analyze, Construct, Execute) framework for a structured analysis.

## Technologies Used
- Python (Pandas, Matplotlib, Seaborn)
- Jupyter Notebook
- Git & GitHub
